# ЛОГИКА КОПИРОВАНИЯ

- Копирование происходит:
   - По клику на целевой элемент на целевой странице в режиме выделения
   - По нажатию на соответствующую кнопку в окне COPIED
- При копировании по клику на элемент:
   1. Выключить режим выделения
   2. Обработка информации:
      1. Для каждого формата (который включён в настройках пользователя) сделать снимок в кэш расширения
      2. Сохранить формат по умолчанию в clipboard (из кэша и только если задан "Copy default")
      - Если обработка занимает более 0.5 секунд с момента клика пользователя, то открыть окно LOADING
      - До окончания обработки данных новый запуск режима выделения-копирования невозможен
   3. Открыть окно COPIED
- При копировании по кнопке из окна COPIED:
   1. Взять соответствующий снимок из кэша расширения
   2. Сохранить результат в clipboard
