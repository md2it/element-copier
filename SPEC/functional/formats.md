## Форматы Download
- Это файлы для скачивания
- Название файла в формате `C-D-T.E`, где:
   - C = `copied`
   - D = домен сайта, точки меняем на дефис -> `-www-` меняем на `-` 
   - T = название тега
   - E = расширение файла
- Список:
   - Markdown // Формируется из текста markdown
   - PNG
   - JPEG
   - HTML // Формируется из текста code
