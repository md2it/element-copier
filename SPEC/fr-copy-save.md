# КОПИРОВАНИЕ

---

## ОБЩАЯ ЛОГИКА
- Копирование происходит в момент нажатия на элемент:
   1. Сначала убрали класс
   2. Потом скопировали
- Когда произошло копирование, то:
   - Расширение выключается
   - Появляется окно "Saved"
- На данный момент копируем в буфер обмена

---

## Окно COPIED
- Эта страница не может быть вызвана из меню
- Содержит:
   - Заголовок "Copied!". Крупный, зелёный, центрованный
   - Текст "[saved format name] saved to your clipboard", [saved format name] жирным
   - [разделительная черта]
   - Inline текст "Other options:" и кнопки для копирования/сохранения в других форматах:
      - Набор кнопок включает все, которые включил пользователь, включая уже скопированный/сохранённый
      - Иконки на кнопках: 
         - copy -- копирование любых данных в clipboard как текст
         - files -- копирование текстовых документов в clipboard как файл
         - images -- копирование изображений в clipboard как файл
         - file-down -- сохранение текстовых документов
         - image-down -- сохранение изображений
   - Так же inline с кнопками есть ссылка на страницу настроек "... change the options"
