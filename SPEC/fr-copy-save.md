# КОПИРОВАНИЕ

---

## ОБЩАЯ ЛОГИКА
- Копирование происходит:
   - По клику на целевой элемент на целевой странице в режиме выделения
   - По нажатию на соответствующую кнопку в окне COPIED
- При копировании по клику на элемент:
   1. Выключить режим выделения
   2. Сразу после выключения выделения делается снимок в кэш раширения всех результатов (только форматы, которые включены у пользователя)
   3. Сохранить формат по умолчанию в clipboard (из кэша и только если задан "Copy default")
   4. Показать окно COPIED
- При копировании по кнопке из окна COPIED:
   1. Взять соответствующую информацию из кэша расширения
   2. Сохранить результат в clipboard
- Очистка кэша расширения:
   - При любом покидании/закрытии окна COPIED
   - При закрытии вкладки
   - При переходе на другую страницу

---

## Окно COPIED
- Эта страница не может быть вызвана из меню
- Содержит:
   - Заголовок "Copied!", крупный, центрованный, зелёный
   - Подзаголовок:
      - "Copied to clipboard: WHAT_COPIED"
      - WHAT_COPIED = то, что скопировано в clopboard
      - Если WHAT_COPIED = nothing, то серая
      - Если WHAT_COPIED != nothing, то чёрная
      - При сохранении в clopboard нового значения, WHAT_COPIED меняется соответственно
   - [разделительная черта]
   - Inline текст "Options:" и кнопки для копирования/сохранения в других форматах:
      - Набор кнопок включает все, которые включил пользователь, включая уже скопированный/сохранённый
      - Иконки на кнопках: 
         - copy -- копирование любых данных в clipboard как текст
         - files -- копирование текстовых документов в clipboard как файл
         - images -- копирование изображений в clipboard как файл
         - file-down -- сохранение текстовых документов
         - image-down -- сохранение изображений
   - Так же inline с кнопками есть ссылка на страницу настроек "... change the options"
