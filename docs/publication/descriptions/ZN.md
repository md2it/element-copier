ELEMENT COPIER
快速、方便地从网页复制和下载内容。

功能
- 复制整页或某个元素，
- 一次将信息转换为多种格式，
- 为所有格式保留最近一次复制的内容，
- 支持快捷键，
- 浅色和深色主题，
- 灵活的设置。

支持的格式
- 用于粘贴到 Google Docs 和 Word 的富文本，
- 图片（PNG、JPEG），
- Markdown，
- 页面源码，
- 面向开发者和测试人员的多种格式：
   - HTML 代码，
   - 选择器，
   - JS 路径，
   - XPath，
   - 完整 XPath，
   - 声明式样式和计算后样式。

安全性
- 扩展不会访问互联网，
- 不会向任何地方传输任何数据。

界面语言
俄语、英语、西班牙语、法语、德语、简体中文、阿拉伯语。

也倾注了心血
- 通过这个扩展进行文本格式化，比直接复制更好（不是完美，但好很多，也方便很多）；
- 快捷键和“默认操作”的组合非常方便，特别是当你需要复制很多内容时，它能加快工作并省心不少；
- 作为开发者，我加入了很多复制工具，对开发和测试非常有帮助。没必要每次都打开 DevTools。如果你不是开发者，而且这些功能对你来说多余，那就在设置里关掉；
- 我在 Markdown 上做了很多工作。甚至连 SVG 图片也会转换并嵌入到代码中，尽量不丢失。还有很多针对版式、链接等方面的改进，以及更多内容。

非常希望这个扩展能对你有帮助 🙌
