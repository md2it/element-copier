## Download formats
- These are downloadable files
- The file name uses the `C-D-T.E` format, where:
   - C = `copied`
   - D = website domain, replace dots with hyphens -> replace `-www-` with `-`
   - T = tag name
   - E = file extension
- List:
   - Markdown // Generated from markdown text
   - PNG
   - JPEG
   - HTML // Generated from code text
